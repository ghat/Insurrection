MKSoft Tests

The following test pages are to test if your browser is compatible with the Insurrection Web Tools for Subversion.  Where ever possible, the tests have been broken out into individual situations such that the core issues can be identified.  The tests require at least and HTML 4.0 browser in order to view the reference samples as they are presented in HTML.

The reference pages are rendered using plain HTML plus style sheets.  The test pages should render just like the reference pages.  The links below will load the two pages within frames in your browser such that you can see the test page and reference page at the same time.
